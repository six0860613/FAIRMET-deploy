## Installation

To get FAIRMET up and running locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/six0860613/FAIRMET-deploy.git

# Navigate into the directory
cd FAIRMET-deploy

# Install dependencies
npm install

# Run locally
npm start
```
